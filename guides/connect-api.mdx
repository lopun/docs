---

Sheetah provides developers with a ready-to-use test template to connect the API to their code. This guide walks you through this process.

After thoroughly testing your template, we suggest copying the auto-generated code snippet from the "Test a Template" documentation under 'Step 3: Ready-to-Use Code Snippet'. At the moment, we only support Node.js, but soon, we will support Python, Java, Kotlin, and Go languages.

![Step 03 test a template](screenshot.jpg)

Here are the steps you should follow:

## 1. Install SDK

First, you need to install our SDK. Run the following command in your terminal:

<CodeGroup>
```bash Node.js
npm install --save sheetah
```
</CodeGroup>

## Step 2. Import Sheetah SDK and initialize API_KEY

After installing the SDK, import it into your project and initialize your API Key.

<CodeGroup>
```javascript Node.js
import Sheetah from 'sheetah'
const sheetah = new Sheetah('YOUR_API_KEY')
```
</CodeGroup>

To create and view your API Key, visit your [developer settings page](https://app.sheetah.io/developer-settings/api-keys).

![API KEY page](apikey_screenshot.jpg)

## Step 3. Connect API

Lastly, you need to connect the API by pasting the code snippet provided at 'Step 3: Ready-to-Use Code Snippet'.

<CodeGroup>
```javascript Node.js
// $ npm install sheetah
// https://www.npmjs.com/package/sheetah
const sheetah = new Sheetah('YOUR_API_KEY')

const request = sheetah
.setTemplateId(64)
.setTables([{
id: "CRMTable",
rows: [
["ABC Company", "John Smith", 5000],
["XYZ Company", "Kevin Choi", 3000]
],
columns: [{
name: "COMPANY NAME",
filter: false,
totalFormula: "none"
}, {
name: "CONTACT NAME",
filter: false,
totalFormula: "none"
}, {
name: "PROJECTED SALES",
filter: true,
totalFormula: "average"
}]
}])
.setVariables({
partner_email: "partner@example.com"
})
.setSheets(undefined)
.setOptions({
filename: "crm-template",
password: "123456",
expireInDays: 7
})

const {
message,
fileUrl
} = await request.exportExcelToFileUrl()

if (message === 'success') {
return fileUrl
} else {
console.log(`Error Message: ${message}`)
}

```
</CodeGroup>

With these steps, you're done! You've now connected your project with the Sheetah API!

For more details on the SDK, please visit: [https://npmjs.com/package/sheetah](https://npmjs.com/package/sheetah).
